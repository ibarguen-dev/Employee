# Employee

Este proyecto fue creado con el fin de mostrar mis capacidades de creación de API con .NET 

## Como se ejecuta

Para poder utilizar este proyecto debemos compilarlos y después ejecutarlos. Para hacer esto debemos seguir los siguientes pasos.

En la parte superior para dos iconos de color verde y una select con el nombre del proyecto primero le damos al botón de bordes verdes esto lo compilara y después lo ejecutara después de que ejecute.


![Barra](/images/BarraHerramientas.png)

 Le damos al botón de  select y colocamos el otro proyecto y volvemos a usar el paso anterior.
